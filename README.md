# Pixel Stretch

Stretch image edges to create seamless tiles, fill backgrounds, or experiment with pixel art.

**Open it:** just download `pixel-stretch.html` and open in your browser. No installation needed.

## What it does

Load an image → drag the slider to stretch one edge → download PNG.

Perfect for:
- **Seamless textures** — extend edges for seamless tiling
- **Background fill** — stretch edge color across transparent areas  
- **Texture prep** — quick edge-stretching for game assets or 3D software
- **Pixelart experiments** — test stretching effects instantly

## Controls

- **AXIS** — Horizontal (H), Vertical (V), or pipe (|)
- **STRETCH** — Which side stretches (→ left/bottom or ← right/top)
- **GRID** — Optional alignment grid (OFF / 4 / 9 / 25 cells)

Drag the slider to adjust where the stretch happens.

## How to Use

1. Open `pixel-stretch.html` in any browser
2. Drop an image or click to select one
3. Drag the slider to set the stretch position
4. Adjust axis, direction, and grid as needed
5. Click **SAVE PNG** to download

**On mobile:** hold the result image to save to Photos.
